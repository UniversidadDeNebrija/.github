# Universidad Nebrija - Escuela Politécnica Superior

Material docente para asignaturas de grado en ingeniería, impartidas por el profesor Carlos Mestre.

## Asignaturas

- **Estructuras de Datos** (Python)
- **Ingeniería del Software** (Java)
- **Programación Avanzada** (Java - concurrencia, patrones de diseño, Swing)
- **Diseño Digital** (VHDL/FPGAs)
- **Sistemas Empotrados** (HTML/docs)

## Organización de los repositorios

Cada repositorio corresponde a una asignatura o bloque temático. Los nombres siguen el patrón `CURSO-Asignatura` o el nombre del tema tratado.
